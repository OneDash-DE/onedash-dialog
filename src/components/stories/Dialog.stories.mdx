# Prompt Dialog

## Basic

A prompt dialog shows a yes or no button. The user has to decide which option he prefers.

```javascript
const PromptDialog = () => {
	const { yesNoPromt } = usePrompt();

	const onClick = () => {
		yesNoPromt("Are you really sure?", "Here comes the text with additional information.").then((yes) => {
			if (yes) {
				console.log("Yes has been pressed");
			} else {
				console.log("No has been pressed");
			}
		});
	};
	return (
		<div className="prompt-dialog">
			<button onClick={onClick}>Open Prompt</button>
		</div>
	);
};
```

Important: You have to wrap the components with "PromptProvider";

```javascript
<PromptProvider>
	<App />
</PromptProvider>
```
